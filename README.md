## Git
コミットメッセージには以下のプレフィックスを付与すること。<br>
| プレフィックス | 説明 |
| ---- | ---- |
| feat: | 新機能 |
| fix: | バグ修正 |
| docs: | ドキュメントのみの変更 |
| format: | コードの動作に影響しない、見た目だけの変更（スペース、フォーマット、セミコロンなど） |
| refactor: | 仕様に影響しないコード改善 |
| perf: | パフォーマンス向上関連 |
| test: | テスト関連 |
| chore: | ビルドプロセス、補助ツール、ライブラリ関連 |
